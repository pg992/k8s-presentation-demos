Azure resources
---------------
az group create --name demo-1 --location WestEurope
az aks create --resource-group demo-1 --name demo-1 --node-count 2 --generate-ssh-keys --kubernetes-version 1.10.12 --node-vm-size Standard_B2s
az aks get-credentials --resource-group demo-1 --name demo-1
az acr create --resource-group demo-1 --name demo-1 --sku Basic --admin-enabled true
az acr login --name demo-1
az aks use-dev-spaces --resource-group demo-1 --name demo-1

Kubernetes dashboard
----------------
kubectl create clusterrolebinding kubernetes-dashboard --clusterrole=cluster-admin --serviceaccount=kube-system:kubernetes-dashboard
az aks browse --resource-group demo-1 --name demo-1
